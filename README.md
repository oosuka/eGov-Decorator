# e-Gov Decorator

e-Gov法令ページ内の全角括弧 `（...）` をハイライト表示する Chrome 拡張です。  
以下のドメインで動作します。

- `https://laws.e-gov.go.jp/*`
- `https://elaws.e-gov.go.jp/*`

## 導入方法

1. このリポジトリをローカルに配置します。
2. Chromeで `chrome://extensions` を開きます。
3. 右上の「デベロッパーモード」を有効にします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックします。
5. このリポジトリのフォルダを選択します。

更新した場合は、拡張機能一覧の「更新」または対象拡張の再読み込みを行ってください。

## ON/OFF の切り替え

- 既定のショートカット: `Ctrl+Shift+Y`
- 押すたびにハイライト表示を ON/OFF します。

ショートカットを変更する場合:

1. `chrome://extensions/shortcuts` を開きます。
2. `e-Gov Decorator` のコマンドに任意のキーを設定します。

## 色の指定方法

ハイライトの背景色と文字色はオプション画面から変更できます。

1. `chrome://extensions` を開きます。
2. `e-Gov Decorator` の「詳細」を開きます。
3. 「拡張機能のオプション」を開きます。
4. 背景色・文字色を選択して「保存」を押します。

「デフォルトに戻す」を押すと以下の色に戻ります。

- 背景色: `#e6e6e6`
- 文字色: `#ffffff`

## 補足

- 設定は `chrome.storage.local` に保存されます。
- ページの動的更新後もハイライトが追従するように実装しています。
